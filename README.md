# AO3-Author-Stats-Tracker-Visualizer
A Python automation tool designed for AO3 authors to track their daily traffic increments (+N) and generate visual trend line charts.
**Features:**
1.Bypass Cloudflare anti-bot protection automatically.
2.Print daily/hourly stat increments (Hits, Kudos, Bookmarks) directly in the terminal.
3.Log data into a CSV file for long-term tracking.
4.Auto-generate a beautiful line chart tracking the growth of each fanfic over time.
